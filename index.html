Problem Statement

Most digital museum platforms provide static information without allowing users to interact naturally. There is a strong need for an intelligent virtual museum guide that enables real-time question answering, typo-tolerant search, and verified responses without hallucination.

Open Knowledge Dataset

This project uses a verified open dataset containing curated museum-based questions and answers. The AI system strictly answers only from this dataset to prevent hallucinations.

View Full Question Dataset

Demo & How It Works

  • Open the web application
  • Select a museum exhibit from the list
  • Ask any question in natural language
  • AI performs fuzzy matching on the dataset
  • Verified answer is returned instantly
  • No hallucinations — trusted responses only

Features

  • Live AI-based Question Answering
  • Fuzzy matching for typo-tolerant queries
  • Mobile & Desktop friendly
  • Scalable & customizable for any museum
  • Open dataset architecture

Example Questions You Can Ask

  • Who painted the Mona Lisa?
  • What is the Rosetta Stone?
  • Which museum has the Veiled Rebecca?
  • Tell me about Tipu Sultan’s Sword
  • When was the Ashokan Lion Capital built?
  • Give me a fun fact about the Dancing Girl

⚠️ The AI answers only questions related to the dataset for factual accuracy.

Technologies Used
