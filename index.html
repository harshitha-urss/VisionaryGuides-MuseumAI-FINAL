Problem Statement

Most digital museum platforms provide only static information without real-time interaction. Visitors cannot ask natural questions or receive instant guidance. This creates a need for an intelligent AI-powered virtual museum guide that allows interactive learning with verified, hallucination-free responses.

System Architecture

  • User selects a museum exhibit
  • User asks a question in natural language
  • AI performs fuzzy search on dataset
  • Matching answer is fetched from question-bank
  • Verified result is displayed instantly
  • No hallucinations – fallback with reference if unknown

View Full Dataset

Working Process

  • Open the web application
  • Choose a museum exhibit from the dropdown
  • Ask any question in your own words
  • AI performs fuzzy matching on dataset
  • Accurate answer is displayed instantly
  • Unknown queries return safe fallback response

Technologies Used

React.js | Node.js | Express.js | Fuzzy Matching AI | HTML | CSS | JavaScript | GitHub | Netlify

Results

The system successfully answers museum-related questions with high accuracy, prevents hallucinations using dataset validation, and delivers fast responses across all devices with excellent reliability.
